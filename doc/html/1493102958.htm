Timer Events

The Timer type exposes the following members.

Events

  Name Description
Completed
Occurs when the timer completed, ticking count equals to the RepeatCount.
Paused
Occurs when call the method Pause()()()().
Reseted
Occurs when call the method Reset()()()().
Resumed
Occurs when call the method Resume()()()().
Started
Occurs when call the method Start()()()().
Stopped
Occurs when call the method Stop()()()().
Ticking
Occurs when the specified timer interval has elapsed.
